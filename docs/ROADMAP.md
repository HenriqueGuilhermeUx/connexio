# Roadmap

## Marco 0 — Protótipo navegável

- [x] identidade inicial Connexio;
- [x] onboarding e cadastro pendente;
- [x] login de demonstração;
- [x] feed, busca e categorias;
- [x] detalhe e WhatsApp;
- [x] favoritos;
- [x] publicação local;
- [x] painel administrativo demonstrativo;
- [x] PRD, arquitetura e modelo de dados.

## Marco 1 — Fundação Supabase

- [ ] criar projeto e ambientes;
- [ ] configurar variáveis de ambiente;
- [ ] autenticação real;
- [ ] schema e migrations;
- [ ] RLS;
- [ ] seed de categorias e regiões;
- [ ] repositórios de dados no app.

## Marco 2 — Operação do catálogo

- [ ] upload e processamento de imagens;
- [ ] criação, edição, pausa e exclusão de anúncios;
- [ ] moderação de anúncios;
- [ ] busca e filtros persistentes;
- [ ] favoritos persistentes;
- [ ] eventos de contato e busca sem resultado.

## Marco 3 — Validação e segurança

- [ ] fila administrativa real;
- [ ] histórico de decisões;
- [ ] suspensão e revalidação;
- [ ] denúncias;
- [ ] recuperação de conta;
- [ ] MFA para administradores;
- [ ] termos, privacidade e exclusão de conta.

## Marco 4 — Piloto fechado

- [ ] curadoria de catálogo inicial;
- [ ] convite de ofertantes por categoria e região;
- [ ] painel de métricas;
- [ ] entrevistas e análise de contatos gerados;
- [ ] correções de retenção e descoberta.

## Depois da validação

Cupons estruturados, notificações, pedidos de indicação, agenda, planos profissionais, destaques e expansão geográfica.
